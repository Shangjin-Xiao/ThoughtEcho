🚀 快速开始

五步上手

简单几步，即可开始使用 ThoughtEcho

1 环境准备

确保已安装 Flutter 3.x+ 环境。运行以下命令检查配置：

flutter doctor

2 获取代码

从 GitHub 克隆项目到本地：

git clone https://github.com/Shangjin-Xiao/ThoughtEcho.git
cd ThoughtEcho

3 安装依赖

安装项目所需的所有依赖包：

flutter pub get

4 运行应用

在模拟器或真机上运行应用：

flutter run

5 AI 功能配置（可选）

在应用设置中配置 API Key 以启用 AI 功能。支持 OpenAI、Anthropic、DeepSeek 等多种 AI 服务商。
