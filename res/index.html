项目概述

Project Overview

不只是记录，更是思想的智能伴侣

Not just a journal, but an intelligent companion for your thoughts

心迹不仅仅是一款简洁的日记应用，更是一个强大的个人思想追踪器和AI洞察助手。它帮助你捕捉每天的灵感、想法和思考，并利用AI技术提供深度分析和洞察，让你能够更好地理解自己的思维模式，获得新的视角和启发。

ThoughtEcho is not just a simple diary app, but a powerful personal thought tracker and AI insight assistant. It helps you capture daily inspirations, ideas, and reflections, and leverages AI technology to provide deep analysis and insights, enabling you to better understand your thinking patterns and gain new perspectives.

基于Flutter框架开发，同时支持移动端和桌面端，随时随地记录和回顾你的心迹。

Developed with Flutter framework, supporting both mobile and desktop platforms, allowing you to record and review your thoughts anytime, anywhere.

核心功能

Core Features

精心设计的功能，为你的思考提供完美支持

Carefully designed features to perfectly support your thinking

✍️
心迹记录
Thought Recording

简洁优雅的界面，让你专注于快速记录想法、感悟、灵感，捕捉每一个心动瞬间。支持富文本编辑，可添加标签和分类。

Clean and elegant interface for focusing on capturing ideas, insights, and inspirations. Supports rich text editing with tags and categories.

✨
AI智能洞察
AI Smart Insights

集成强大的AI服务，深度分析你的笔记内容，提供情感分析、核心思想提炼、行动建议和思维模式分析等多维度智能洞察。

Integrated with powerful AI services to analyze your notes and provide emotional analysis, core idea extraction, action suggestions, and thinking pattern analysis.

🤔
每日思考提示
Daily Thought Prompts

每天为你生成一个发人深省的哲学或智慧提示，激发深度思考，拓展思维边界，培养反思习惯。

Daily philosophical or wisdom prompts to inspire deep thinking, expand your mental boundaries, and cultivate reflection habits.

🗂️
分类管理
Category Management

自定义笔记分类，灵活组织和管理你的心迹，方便回顾和检索。支持自定义分类图标，视觉化管理更直观。

Customize note categories for flexible organization and management. Support custom category icons for intuitive visual management.

🏷️
标签系统
Tag System

为笔记添加多个标签，实现多维度归类，提升检索效率，构建知识网络。

Add multiple tags to notes for multi-dimensional categorization, improving retrieval efficiency and building knowledge networks.

🔍
关键词搜索
Keyword Search

强大的关键词搜索功能，快速定位所需笔记，支持标题、内容、标签多维度搜索，回顾特定时期的思考。

Powerful keyword search to quickly locate notes, with multi-dimensional search across titles, content, and tags.

📍
位置记录
Location Tracking

自动或手动记录笔记创建位置，为思考增添空间维度，回忆更加丰富立体。

Automatically or manually record the location where notes are created, adding a spatial dimension to your thoughts.

☁️
天气记录
Weather Recording

自动记录笔记创建时的天气状况，为日后回顾增添环境背景，丰富记忆线索。

Automatically record weather conditions when creating notes, adding environmental context for future review.

🌙
深色模式
Dark Mode

提供舒适的深色模式，呵护你的双眼，适应不同光线环境，夜间使用更护眼。

Comfortable dark mode to protect your eyes and adapt to different lighting environments.

用户安装指南

User Installation Guide

  1. 下载应用：从GitHub发布页面下载最新版本
  2. Download the app: Get the latest version from the GitHub release page
  3. 首次启动设置：安装完成后打开应用，根据引导完成初始化设置
  4. First launch setup: Open the app after installation and follow the guide to complete initial setup
  5. 开始使用：点击主页面右下角的「+」按钮创建新笔记，在设置页面自定义应用主题和功能偏好
  6. Start using: Click the "+" button in the bottom right corner to create new notes, customize app theme and preferences in the settings page

开发者安装指南

Developer Installation Guide

  1. 安装Flutter SDK：请确保你的开发环境已安装Flutter SDK
  2. Install Flutter SDK: Ensure your development environment has the Flutter SDK installed
  3. 克隆项目 Clone the project 
    git clone https://github.com/Shangjin-Xiao/ThoughtEcho.git
cd ThoughtEcho
  4. 安装依赖 Install dependencies 
    flutter pub get
  5. 运行应用 Run the app 
    flutter run
