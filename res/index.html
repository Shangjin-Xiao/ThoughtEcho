🚀 快速开始 🚀 Quick Start

五步上手 Get Started in 5 Steps

简单几步，即可开始使用 ThoughtEcho Simple steps to start using ThoughtEcho

1 环境准备Prerequisites

确保已安装 Flutter 3.x+ 环境。运行以下命令检查配置：Ensure Flutter 3.x+ is installed. Run this command to check:

flutter doctor

2 获取代码Get the Code

从 GitHub 克隆项目到本地：Clone the project from GitHub:

git clone https://github.com/Shangjin-Xiao/ThoughtEcho.git
cd ThoughtEcho

3 安装依赖Install Dependencies

安装项目所需的所有依赖包：Install all required dependencies:

flutter pub get

4 运行应用Run the App

在模拟器或真机上运行应用：Run on emulator or real device:

flutter run

5 AI 功能配置（可选）AI Configuration (Optional)

在应用设置中配置 API Key 以启用 AI 功能。支持 OpenAI、Anthropic、DeepSeek 等多种 AI 服务商。Configure API Key in app settings to enable AI features. Supports OpenAI, Anthropic, DeepSeek, and more.
