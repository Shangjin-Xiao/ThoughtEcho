项目概述

心迹不仅仅是一款简洁的日记应用，更是一个强大的个人思想追踪器和AI洞察助手。它帮助你捕捉和记录日常思绪，并利用先进的AI技术，深入分析你的笔记内容，挖掘潜在的模式、情感和洞察，从而促进自我认知、提升思考深度，实现个人成长。

基于Flutter框架开发，同时支持移动端和桌面端，随时随地记录和回顾你的心迹。

核心功能

✍️
心迹记录

简洁优雅的界面，让你专注于快速记录想法、感悟、灵感，捕捉每一个心动瞬间。支持富文本编辑，可添加标签和分类，轻松整理思绪。

🧠
AI智能洞察

集成强大的AI服务，深度分析你的笔记内容，提供情感分析、核心思想提炼、行动建议和思维模式分析等多维度智能洞察。

🤔
每日思考提示

每天为你生成一个发人深省的哲学或智慧提示，激发深度思考，拓展思维边界，培养反思习惯。

🗂️
分类管理

自定义笔记分类，灵活组织和管理你的心迹，方便回顾和检索。支持自定义分类图标，视觉化管理更直观。

🏷️
标签系统

为笔记添加多个标签，实现多维度归类，提升检索效率，构建知识网络。

🔍
关键词搜索

强大的关键词搜索功能，快速定位所需笔记，支持标题、内容、标签多维度搜索，回顾特定时期的思考。

📍
位置记录

自动或手动记录笔记创建位置，为思考增添空间维度，回忆更加丰富立体。

☁️
天气记录

自动记录笔记创建时的天气状况，为日后回顾增添环境背景，丰富记忆线索。

🌙
深色模式

提供舒适的深色模式，呵护你的双眼，适应不同光线环境，夜间使用更护眼。

应用界面

开始使用

用户安装指南

  1. 下载应用：从GitHub发布页面下载最新版本
  2. 首次启动设置：安装完成后打开应用，根据引导完成初始化设置
  3. 开始使用：点击主页面右下角的「+」按钮创建新笔记，在设置页面自定义应用主题和功能偏好

开发者安装指南

  1. 安装Flutter SDK：请确保你的开发环境已安装Flutter SDK
  2. 克隆项目： 
    git clone https://github.com/Shangjin-Xiao/ThoughtEcho.git
cd ThoughtEcho
  3. 安装依赖： 
    flutter pub get
  4. 运行应用： 
    flutter run
