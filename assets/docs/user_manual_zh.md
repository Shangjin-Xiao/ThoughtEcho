# 心迹 (ThoughtEcho) 用户手册

欢迎使用心迹，您的专属 AI 灵感笔记本。本手册将帮助您快速了解应用的核心功能。

> 📖 **完整手册**：访问 [在线文档](https://shangjin-xiao.github.io/ThoughtEcho/user-guide.html) 查看详细指南

## 1. AI 服务配置

### 支持的 AI 服务商
- **OpenAI**（gpt-4o）
- **Anthropic Claude**（claude-3.7-sonnet-latest）
- **DeepSeek**（deepseek-chat）
- **OpenRouter**（多模型聚合）
- **SiliconFlow**（国内服务）
- **Ollama**（本地开源模型）
- **LMStudio**（本地推理）
- 其他兼容 OpenAI API 的服务

### 快速配置步骤
1. 打开「设置」→「AI 助手设置」
2. 选择或添加服务商
3. 填入 API Key（加密安全存储）
4. 点击「测试连接」验证
5. 保存设置

**本地 AI（免费）**：安装 Ollama 或 LMStudio，下载模型后配置即可，无需 API Key。

## 2. 富文本编辑器

### 工具栏功能
- **基础样式**：加粗、斜体、下划线、删除线
- **标题格式**：H1-H6 多级标题
- **文字控制**：字号、字体、颜色、对齐
- **列表布局**：有序/无序列表、缩进、引用块、代码块
- **媒体插入**：图片、视频、音频

### AI 辅助功能（✨ 按钮）
- **智能分析来源**：自动识别作者和出处
- **润色文本**：改进表达和文字风格
- **AI 续写**：继续你的思路
- **深度分析**：生成笔记摘要和洞察
- **问笔记**：针对内容与 AI 对话

### 自动功能
- 每 2 秒自动保存草稿
- 位置和天气自动获取
- 媒体文件引用管理
- 大文件内存优化

## 3. AI 功能

### 每日灵感
根据时间、天气、历史笔记生成个性化写作提示。

### 周期报告
- **周报**：过去 7 天统计
- **月报**：过去 30 天统计
- **年报**：全年数据回顾
- AI 生成的诗意洞察
- 精选笔记卡片

### AI 洞察分析
**分析类型**：综合分析、情感分析、思维导图、成长分析  
**分析风格**：专业、友好、幽默、文学

### 年度报告
生成精美的 HTML 年度总结，包含统计、AI 洞察、数据可视化。

## 4. 笔记管理

### 排序与筛选
- **排序**：按时间、名称、喜爱度
- **筛选**：按标签、分类、天气、时间段、情感

### 笔记操作
- **左滑删除**：快速删除笔记
- **点击爱心**：增加喜爱度
- **分享**：纯文本或精美卡片（15+ 模板）
- **全屏编辑**：进入富文本编辑器
- **AI 分析**：生成洞察

### 搜索
全文搜索（标题 + 内容），500ms 防抖优化，实时高亮。

## 5. 多设备同步

### LocalSend 协议
无需云服务器的局域网 P2P 同步。

### 同步流程
1. 自动发现同一 WiFi 下的设备
2. 选择目标设备
3. 发送/接收数据
4. 使用"最后写入者胜"策略自动合并

### 支持平台
Android、iOS、Windows、macOS、Linux（Web 受限）

## 6. 备份与恢复

### 备份格式
- **ZIP 格式**（推荐）：包含笔记和媒体文件
- **JSON 格式**：兼容旧版本

### 备份特点
- 流式导出，防止内存溢出
- 分块处理，实时进度显示
- 内存监控，智能中止

### 恢复模式
- **覆盖**：清空后导入（设备重置）
- **合并**：智能合并数据（多设备同步）
- **追加**：简单追加（兼容模式）

**访问路径**：设置 → 备份与恢复

## 8. 常见问题

**Q: AI 功能无法使用？**  
A: 检查 AI 设置中的 API Key，使用「测试连接」验证。

**Q: 如何保护隐私笔记？**  
A: 使用隐藏标签 + 生物识别保护。

**Q: 同步失败？**  
A: 确保两台设备在同一网络，关闭防火墙/VPN 后重试。

**Q: 本地 AI 如何使用？**  
A: 安装 Ollama（ollama.ai）或 LMStudio（lmstudio.ai），下载模型后在 AI 设置中配置。

**Q: 备份文件在哪里？**  
A: 保存在您选择的位置，文件名：`thoughtecho_backup_[时间戳].zip`
