# 心迹 (ThoughtEcho) 用户手册

欢迎使用心迹，您的专属 AI 灵感笔记本。本手册将帮助您快速了解应用的核心功能。

## 1. 富文本编辑器
- **多媒体支持**：点击工具栏图标可插入图片、音频和视频。
- **自动保存**：编辑过程中会自动保存草稿，防止内容丢失。
- **元数据记录**：新建笔记时可勾选自动附加位置和天气信息。
- **大文件保护**：针对超大文档和媒体文件进行了内存优化，确保使用流畅。

## 2. AI 助手
- **笔记问答**：在笔记详情页点击 AI 图标，可以针对该笔记内容进行深度问答。
- **智能分析**：在工具栏中可以使用润色、续写、来源分析等功能。
- **周期洞察**：在“洞察”页面，AI 会分析您的写作趋势、情感变化并生成诗意洞察。
- **年度报告**：支持生成精美的 AI 年度总结报告。

## 3. 多设备同步
- **本地局域网同步**：基于 LocalSend 协议，支持在同一 WiFi 下的设备间快速同步。
- **LWW 合并策略**：使用“最后写入者胜”策略自动合并多端笔记，保留最新修改。
- **跨平台支持**：支持 Android、iOS、Windows 和 Web 平台。

## 4. 备份与恢复
- **ZIP 流式备份**：支持将所有笔记和媒体文件打包为 ZIP 格式，采用流式处理技术防止内存溢出。
- **旧版兼容**：兼容导入旧版本的 JSON 格式备份。
- **合并导入**：支持将备份数据与当前数据合并，而不是覆盖。

## 5. 搜索与分类
- **智能搜索**：支持全文搜索，并带有 500ms 的输入防抖处理，性能高效。
- **标签管理**：通过标签组织笔记，支持隐藏标签以保护私密内容。
- **筛选排序**：可按时间、喜爱度、天气、位置等维度筛选笔记。
