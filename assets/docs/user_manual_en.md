1. Open "Settings" â†’ "AI Assistant Settings"

### LocalSend Protocol
LAN P2P sync without cloud servers.

### Sync Flow
1. Auto-discover devices on same WiFi
2. Select target device
3. Send/receive data
4. Auto-merge using "Last Write Wins" strategy

## 8. FAQ

**Q: AI features not working?**  
A: Check API Key in AI settings, use "Test Connection" to verify.

**Q: How to protect private notes?**  
A: Use hidden tags + biometric protection.

**Q: Sync failing?**  
A: Ensure both devices on same network, disable firewall/VPN and retry.

**Q: How to use local AI?**  
A: Install Ollama (ollama.ai) or LMStudio (lmstudio.ai), download models, then configure in AI settings.

**Q: Where are backup files?**  
A: Saved to your chosen location, filename: `thoughtecho_backup_[timestamp].zip`
