# ThoughtEcho User Manual

Welcome to ThoughtEcho, your personal AI-powered inspiration notebook. This manual will help you understand the core features of the app.

## 1. Rich Text Editor
- **Multimedia Support**: Tap toolbar icons to insert images, audio, and video.
- **Auto-save**: Drafts are automatically saved during editing to prevent data loss.
- **Context Capture**: Optionally auto-attach location and weather info when creating new notes.
- **Resource Protection**: Memory optimization for large documents and media files ensures a smooth experience.

## 2. AI Assistant
- **Note Q&A**: Tap the AI icon in note details to ask questions specifically about that note.
- **Smart Tools**: Use AI to polish text, continue writing, or analyze potential sources (author/work).
- **Periodic Insights**: In the "Insights" tab, AI analyzes your writing trends and emotional changes to generate poetic reflections.
- **Annual Report**: Generate beautiful AI-powered summaries of your year.

## 3. Multi-device Sync
- **LAN Sync**: Powered by the LocalSend protocol, enabling fast synchronization between devices on the same WiFi.
- **LWW Merge Strategy**: Automatically merges notes using "Last Write Wins" logic to keep the most recent edits.
- **Cross-platform**: Available on Android, iOS, Windows, and Web.

## 4. Backup & Restore
- **Streaming ZIP Backup**: Package all notes and media files into a ZIP archive using streaming technology to prevent memory issues.
- **Legacy Compatibility**: Supports importing JSON backups from older versions.
- **Merge Import**: Optionally merge backup data with existing records instead of overwriting.

## 5. Search & Categories
- **Smart Search**: Full-text search with 500ms debouncing for efficient performance.
- **Tag Management**: Organize notes with tags and use hidden tags for private content.
- **Filters & Sorting**: Filter notes by time, favorites, weather, location, and more.
