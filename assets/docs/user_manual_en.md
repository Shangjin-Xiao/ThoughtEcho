- **SiliconFlow** (Domestic AI service)
- **Ollama** (Run AI for free on your own computer)
- **LMStudio** (Local AI inference)
- Other compatible services

### Quick Setup
1. Open "Settings" â†’ "AI Assistant Settings"
2. Select or add a provider
3. Enter API Key (saved safely on your device)
4. Click "Test Connection" to verify
5. Save settings

**Local AI (Free)**: Install Ollama or LMStudio, download models, then configure. No API Key needed.

## 5. Multi-device Sync

### Sync Method
Directly sync between devices on the same WiFi network, no cloud server required.

### Sync Flow
1. The app will look for your other devices automatically on the same WiFi
2. Select target device
3. Send/receive data
4. Intelligently merge notes (The app keeps your most recent changes)

### Supported Platforms
Android, iOS, Windows, macOS, Linux (Web limited)

## 8. FAQ

**Q: AI features not working?**  
A: Check API Key in AI settings, use "Test Connection" to verify.

**Q: How to protect private notes?**  
A: Use hidden tags + biometric protection.

**Q: Sync failing?**  
A: Ensure both devices on same network, disable firewall/VPN and retry.

**Q: How to use local AI?**  
A: Install Ollama (ollama.ai) or LMStudio (lmstudio.ai), download models, then configure in AI settings.

**Q: Where are backup files?**  
A: Saved to your chosen location, filename: `thoughtecho_backup_[timestamp].zip`
